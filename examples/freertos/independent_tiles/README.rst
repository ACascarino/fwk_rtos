****************************
Running the project firmware
****************************

Begin by flashing the WiFi module firmware and setting up the WiFi network:

.. code-block:: console

    $ ./flash_image.sh


To build and run the firmware, run the following commands:

.. code-block:: console

    $ make
    $ make run
